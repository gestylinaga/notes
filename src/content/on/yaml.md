# YAML

***WorkInProgress***

**YAML** (**Y**AML **A**in't **M**arkup **L**anguage) is a data interchange 
format, with syntax based on [Python's 🗒️](/on/python/) indentation style.

## Syntax
- YAML files use the file ***extensions*** `.yml` or `.yaml`

## Example
```yaml
--- # example yaml
first_name: Gesty
last_name: Linaga
```
