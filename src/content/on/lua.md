# Lua
```lua 
#!/usr/bin/env lua 

local function helloWorld()
  io.write("This is Lua")
end

helloWorld()
```
**Lua** is a fast, lightweight, and embeddable scripting/programming language. 
It is widely considered the *fastest* interpreted/scripting language.
